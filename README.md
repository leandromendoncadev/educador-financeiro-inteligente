# Educador Financeiro Inteligente 🤖💰

Projeto desenvolvido como desafio criativo da DIO.

## Sobre o projeto

Proposta de um Educador Financeiro Inteligente utilizando conceitos de:

- Inteligência Artificial
- UX/UI
- Desenvolvimento de produtos digitais
- Personalização de experiências

## Objetivo

Criar uma solução capaz de auxiliar usuários no controle financeiro,
organização de gastos e criação de hábitos financeiros saudáveis.

## Conceito

O sistema funciona como um mentor financeiro digital,
oferecendo recomendações personalizadas, organização financeira
e educação sobre dinheiro de forma simples e acessível.
